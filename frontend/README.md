# Dark Pattern Lab Frontend

This folder contains the React + Vite frontend for Dark Pattern Lab.

It is the participant-facing part of the dissertation artefact. Participants use it to move through the pre-test step, complete the dark pattern simulations, view their results, and continue to the post-test survey.

## Useful Commands

```bash
npm install
npm run dev
npm run lint
npm run build
```

The main project documentation is in the root `README.md`.
